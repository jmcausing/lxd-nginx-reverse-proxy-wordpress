# lxd-nginx-reverse-proxy-wordpress
WordPress site generator using LXD + Nginx Reverse Proxy

This bash script will create a WordPress site using:

- LXD
- Nginx reverse proxy container
- Cloudflare for subdomain
- Let's Encrypt SSL
- Install WP-CLI
